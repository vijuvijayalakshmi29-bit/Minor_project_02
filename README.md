# 💳 SpendDNA

SpendDNA is a Python-based project that analyzes bank and UPI transaction data to provide meaningful spending insights. It cleans transaction data, identifies merchants, categorizes expenses, and generates reports to help users understand their financial habits.

## Features

* Transaction Parsing & Data Cleaning
* Vendor (Merchant) Normalization
* Expense Categorization
* Monthly Spending Analysis
* Time-Based Spending Insights
* Anomaly Detection
* Financial Summary Report

## Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Regular Expressions (Regex)
* Jupyter Notebook

## Getting Started

1. Clone this repository.
2. Install the required libraries.
3. Open the notebook and run all cells.

## Project Goal

The goal of SpendDNA is to convert raw transaction data into clear and useful financial insights using Python and data analysis techniques.
